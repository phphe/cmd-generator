pwd

pwd 命令用于显示当前工作目录的路径。
$

例子

  • 显示当前工作目录的路径：pwd
  • 显示当前工作目录的绝对路径：pwd -P
  • 显示当前工作目录的逻辑路径（不跟随符号链接）：pwd -L
  • 显示当前工作目录的物理路径（跟随符号链接）：pwd -P -L
